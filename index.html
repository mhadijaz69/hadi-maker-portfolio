Featured Project

✈️ Flight Fuel & Emissions Calculator

An interactive web application developed using Python and Streamlit that estimates aircraft fuel consumption, carbon dioxide emissions, passenger impact, and operating costs using published aviation regression models and ICAO emission factors.

Current features include:

  • Flight distance calculation between airports
  • Comparison of multiple commercial aircraft
  • Automatic passenger estimation
  • Fuel cost estimation using current aviation fuel prices
  • Aircraft range validation
  • Commercial airline fleet selection
  • CO₂ emissions calculation
  • Interactive comparison charts
