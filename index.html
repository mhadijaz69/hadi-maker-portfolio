✈️ Featured Engineering Project

Flight Fuel & Emissions Calculator

💡 Inspiration

The aviation industry is responsible for a significant share of global carbon dioxide emissions, yet many people have little understanding of how different aircraft, airlines, and routes influence environmental impact. I wanted to create a tool that makes these concepts easy to explore using real aviation data.

🎯 The Challenge

Many online flight calculators provide only basic estimates without explaining why emissions differ between aircraft, airlines, or flight routes. I wanted to design an application that not only performs calculations but also helps users understand the environmental impact of aviation in a simple and interactive way.

⚙️ My Solution

To address this challenge, I developed a web application using Python and Streamlit. The application estimates flight distance, fuel consumption, carbon dioxide emissions, operating fuel costs, and passenger impact while allowing users to compare different commercial aircraft on the same route.

To achieve this, I developed a web application using Python and Streamlit that estimates flight distance, fuel consumption, carbon dioxide emissions, operating fuel cost, and cost per passenger. The application allows users to compare different commercial aircraft and determine which option is more environmentally and economically efficient for a specific route.

Rather than relying on simple assumptions, the calculator incorporates published aviation research, ICAO emission factors, aircraft performance data, airport databases, and airline fleet information to produce realistic estimates while remaining easy for non-technical users to understand.

Key Features

  • Compare multiple commercial aircraft on the same route.
  • Calculate flight distance using real airport coordinates.
  • Estimate aviation fuel consumption.
  • Predict CO₂ emissions using ICAO emission factors.
  • Estimate operating fuel costs.
  • Calculate passenger cost contribution.
  • Verify whether an aircraft can complete a selected route based on its maximum range.
  • Display aircraft specifications including seating capacity, cruise speed, engines, and range.
  • Generate visual comparison charts for emissions and operating costs.
